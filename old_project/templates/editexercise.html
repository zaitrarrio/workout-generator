CHECK OUT THIS CRAZINESS: The alternate workout components were added in WAY AFTER the initial design of the project, so it's more or less a duct tape solution. If you add alternate workout components, the resulting workouts MIGHT NOT BE INTUITIVE TO YOU. The end workout will display the component as indicated by THE FIRST WORKOUT COMPONENT. So for debugging purposes OR if you want a certain order to be maintained in the workout, make 'balance' or 'reactive' the first workout component.

This should be fairly straightforward, but a couple of things: Isolated/compound is a very subjective term, but the program basically ensures that compound exercises are present if a muscle group is worked. This is to prevent a chest workout of nothing but flyes and cable crossovers to happen, which would be incredibly lame. And this website will not be lame, I refuse. So make sure that even if you have something that just works triceps but it's still a "mass builder" (for lack of a better term), make sure and tag it as compound. A better way to think of it might be primary/secondary exercise.

Also too, whatever phases are checked, those are the only phases that this exercise will appear in. Tagging by workout component puts the exercise in a pool of all other exercises within the same workout component. Frameworks for the week are built around workout components, then volumes determine the amount of exercises per workout component, then exercises of that same workout component are selected.

There is a minimum difficulty, but no max difficulty. So the higher your fitness level, the bigger the pool of exercises to choose from.

Oh yeah, this is important...for the URL, don't actually type in a URL. Instead, use the ID of the youtube video. So for something like 'http://www.youtube.com/watch?v='aqSkj8dhsyQ' just put in 'aqSkj8dhsyQ' as the URL. It makes for a lot less work for me if things are done like that.

If an exercise is done one limb at a time, this is for trimming a workout to time. If an exercise is timed instead of repped, exercises will appear in today's workout as '3 x Plank' or whatever instead of '3x10' plank. From there, guidance about how long to do the exercise can be present on the video itself with youtube captions.

Finally, 'don't use the following muscle groups tomorrow'...there's been some debate about that term, but that's basically all the program uses that functionality for at the moment. So don't tag triceps as a secondary exercise to benchpress; triceps could still be worked the following day. The problem comes when we have squats one day and then lunges the very next day. The important thing is really that you are consistent with your tagging, because muscle groups are only used to determine what exercises can be appear where within the week.
