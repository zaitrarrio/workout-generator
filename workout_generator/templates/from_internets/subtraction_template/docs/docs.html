Getting Started

After unzipping the file you've downloaded you will have all the templates you need to get started (index.html, about.html, contact.html etc.). You can edit the content of these files in any text editor or IDE you may want to use.

Once you've edited in your own content simply upload all files and folders to your web server.

Contact page/form

First thing you're going to want to do is open up the "submit.php" file. Look for this line of code at the top:

                	$send_to = "enter@your.email";

Here you want to enter your own email address in between the quotation marks. This next line is optional. It will be what shows up as the subject in your inbox when you receive and email from the contact form. Once again edit in between the quotation marks:

                	$send_subject = "Contact Form ";

Twitter/Flickr

Due to recent Twitter API changes there a few extra steps to take, involving setting up a Twitter Application, to get your feed setup. Start by going to https://dev.twitter.com/apps/. You do not have to sign with the twitter name you are going to use for the feed (or feeds). Once you sign in click "Create new application". Enter any details you want. They do not have to be related to the feed you want to use. Click "Create". On the next screen click "Create my access token". Mark down your Consumer key, Consumer secret, Access token and Access token secret. In the file labeled get_tweets.php look for the following lines of code and enter the values accordingly. (Leave $notweets as is. You can change the number of tweets displayed in "twitter_feed.js")

					$twitteruser = "";
					$notweets = 30;
					$consumerkey = "";
					$consumersecret = "";
					$accesstoken = "";
					$accesstokensecret = "";

Where it says $twitteruser enter you twitter handle between the quotation marks.

To change the number of tweets displayed open up the "twitter_feed.js" file and on line 3 you'll find "var displaylimit". Change that value to how many tweets you want displayed.

For Flickr open up "scripts.js" and go to the section below and on the line for id enter your flickr ID (you can find it by going to your flickr account) where it says "your_id_here":

								  /* Flickr feed */
								  $('.flickr').jflickrfeed({
								    limit : 8,
								    qstrings : {
								      id : 'your_flickr_id_here'
								    },
								    itemTemplate : '
  • ' + '' + '
    • '
								  });

    Tool Tips

    You can add a tool tip to any element by adding a class of "tool" to it. Then add a title attribute containing whatever text you want in the tool tip.

    Google Maps

    In the scripts file towards the bottom you will find where the Google Maps plugin is initiated. There will be an option labeled "address:". There you can enter an address or Ciy, State etc.
